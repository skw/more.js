# more.js (WiP)

more.js is a conversion tool for LESS to other CSS languages, such as Styl, Stylus et al. 

It does not parse LESS directly but utilizes the LESS css parser to extract a JS object and build a css object.

## Features

LESS to: 

- Styl
- Stylus

## Complete:

- selector conversion
- property conversion
- variable support
